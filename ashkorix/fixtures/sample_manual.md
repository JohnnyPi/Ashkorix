# Compliance Manual Excerpt

## Chapter 1 — Overview

This manual describes verification and assessment procedures for control objectives.

## Chapter 2 — Field Reference

### Field 04600 — Compliance flag

Field 04600 indicates whether traceability requirements are met. Error 4042 may appear if validation fails.

### Field 04601 — Audit trail

Related to compliance verification and shall be reviewed quarterly.

| Code | Description |
